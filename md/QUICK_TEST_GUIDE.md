# EXAONE + RAG 빠른 테스트 가이드

프론트엔드 챗봇에서 정책 기반 질문(ADVICE)에 대해 학습된 EXAONE 모델과 Neon DB RAG가 작동하는지 빠르게 테스트하는 방법입니다.

## 🚀 빠른 시작

### 1. 백엔드 서버 실행

```bash
cd backend
python -m uvicorn main:app --reload --port 8000
```

### 2. 프론트엔드 실행

```bash
cd frontend
npm run dev
```

### 3. 브라우저에서 테스트

1. `http://localhost:3000` 접속
2. 챗봇 입력창에 다음 질문 입력:

```
직장인인데 하루 4시간 공부로 합격 가능할까?
```

3. 응답 확인:
   - ✅ 학습된 EXAONE 모델의 말투로 답변
   - ✅ Neon DB에서 검색한 합격 수기 내용 포함
   - ✅ 따뜻하고 전문적인 어조

## 🧪 테스트 질문 예시

다음 질문들을 입력해보세요:

### 학습 상담 질문 (ADVICE 의도)

1. **일일 계획**
   - "전업 수험생으로 공부하는데 하루 일과를 어떻게 구성해야 할까요?"
   - "직장인인데 하루 4시간 공부로 합격 가능할까?"

2. **과목별 학습법**
   - "국어 학습 전략이 궁금해요. 합격자들의 조언을 듣고 싶어요."
   - "영어 노베이스인데 어떻게 시작해야 할까요?"

3. **수험 기간**
   - "1년~1년 6개월 수험기간으로는 부족한가요?"
   - "언제부터 시작해야 할까요?"

4. **어려움 극복**
   - "공부하다가 지칠 때 어떻게 해야 할까요?"
   - "동기부여가 안 될 때는 어떻게 해야 하나요?"

## ✅ 성공 확인

응답에 다음이 포함되어야 합니다:

1. **학습된 말투**
   - "전업 수험생이시군요!"
   - "물론 가능합니다!"
   - "합격하신 선배님의 조언을 들려드릴게요."

2. **합격 수기 내용**
   - 실제 합격자들의 경험담
   - 구체적인 학습 방법
   - 시험 정보 (연도, 직렬, 등급)

3. **Chain-of-Thought 방식**
   - 논리적인 설명
   - 단계별 조언

## 🔍 작동 흐름

```
사용자 질문 입력
    ↓
프론트엔드: /api/chat
    ↓
백엔드: /api/v1/admin/exam/flow
    ↓
게이트웨이 분류: POLICY_BASED
    ↓
의도 분류: ADVICE
    ↓
ExamAgent._handle_advice_with_rag()
    ↓
1. Neon DB 벡터 검색 (KURE-v1)
    → 관련 합격 수기 3개 검색
    ↓
2. EXAONE 모델 응답 생성
    → 학습된 LoRA 어댑터 사용
    → 합격 수기 기반 답변 생성
    ↓
프론트엔드에 응답 반환
```

## 🐛 문제 해결

### 문제 1: "관련 합격 수기를 찾지 못했습니다"

**원인**: Neon DB에 데이터가 없음

**해결**:
```bash
conda activate torch313
python create_embeddings_for_neon.py
```

### 문제 2: "LoRA 어댑터 경로를 찾을 수 없습니다"

**원인**: 학습된 LoRA 어댑터가 없음

**해결**:
- 학습이 완료되었는지 확인
- `artifacts/lora-adapters/exaone-success-stories/` 경로 확인

### 문제 3: 응답이 느림

**원인**: 첫 요청 시 모델 로드 시간 필요

**해결**:
- 첫 요청: 약 30-60초 (모델 로드)
- 이후 요청: 약 5-10초 (캐시된 모델 사용)

### 문제 4: 응답 형식이 이상함

**원인**: 프론트엔드에서 ADVICE 응답 처리 오류

**해결**:
- `frontend/app/api/chat/route.ts` 확인
- `answer.response` 필드 확인

## 📊 로그 확인

백엔드 서버 로그에서 다음을 확인:

```
[ExamFlow] POLICY_BASED → 의도 분류 후 처리
[ExamFlow] ADVICE → exam_agent로 라우팅 (LLM 사용)
[SuccessStoriesRAG] KURE-v1 모델 로딩 중...
[SuccessStoriesRAG] 검색 완료: 3개 결과
[ExaoneLLM] 모델 로드 시작: artifacts/base-models/exaone
[ExaoneLLM] LoRA 어댑터 로드 시작: artifacts/lora-adapters/exaone-success-stories
[ExaoneLLM] LoRA 어댑터 로드 완료
[ExamAgent] ADVICE 의도 감지 → RAG + EXAONE 사용
```
